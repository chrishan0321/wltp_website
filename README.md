# 임페리움 서버 웹

임페리움 서버를 위한 Django 서버입니다.

## 🚀 프로젝트 개요

이 프로젝트는 Django를 사용하여 구축된 임페리움 서버 웹입니다. 플레이어 정보, 랭킹, 공지사항, 갤러리 등의 데이터를 조회할 수있습니다


## 🎨 프론트엔드 기능

### 주요 기능
- **반응형 디자인**: 모바일과 데스크톱 최적화
